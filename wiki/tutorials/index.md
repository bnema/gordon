## Prerequisites

Before starting tutorials, you should have:

1. Gordon installed on a VPS
2. Docker or Podman on your local machine
3. Domain pointing to your VPS
4. Cloudflare (or similar) for HTTPS

See [Installation Guide](/docs/installation.md) if you haven't set up Gordon yet.
